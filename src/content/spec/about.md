你好，我是 **erode**。

这个昵称来自QQ时代——初二那年给自己取了「侵蚀Emotional°」，此后十多年再没换过。后来把侵蚀的英文 erode 用作了微信和所有社交平台的名字。

侵蚀是慢的，但不代表没有力量。水滴石穿，后发制人，这是我认同的节奏。Emotional 是真的，只是不轻易示人。

### 关于我

计算机硕士在读，MBTI 是 INTJ，白羊座——习惯先想清楚「为什么」再动手，追求深度理解而非表面掌握。

我信**复利**。不追求一夜精通，希望每一次提交、每一篇笔记都是未来的积累。

目前正处于从学生思维向工程师思维转变的阶段：在真实的工程中打磨实战能力，用项目倒逼成长。

### 正在学习

主要关注 **AI 应用开发**方向——LLM、RAG、Agent，以及它们在真实业务场景中的落地。比起追前沿论文，研究算法，我更喜欢把技术做成能跑的东西。

我的学习方式：

1. **先跑通，再优化**：从 MVP 开始，不追求一步到位。
2. **公开记录**：通过 GitHub 和这个博客记录迭代过程，倒逼自己输出。
3. **构建知识体系**：放弃所有笔记软件，用文件夹 + Claude Code 作为我的知识管理工具。
